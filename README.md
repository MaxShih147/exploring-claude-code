# Exploring Claude Code

本 repository 是一份 **從全新 macOS 使用者帳號開始**，  
一步一步探索並建立 **Claude Code 開發環境** 的實作紀錄與教學素材。

這不是速成教學，而是一條 **可重現、可教學、真的照做會成功** 的探索路線，  
特別為 **沒有程式背景，或第一次接觸 CLI / Git / AI 工具的新手** 所設計。

---

## 這個 repo 在做什麼？

- 記錄在「乾淨的 macOS 帳號（student）」中實際走過的每一個步驟  
- 整理成可直接使用的 Markdown 教學文件  
- 作為未來 demo、內訓、onboarding 的基礎教材

重點不是炫技，而是回答一個問題：

> **「如果今天是一個完全沒寫過程式的人，要怎麼真的開始用 Claude Code？」**

---

## 教學內容結構

目前內容依照實際操作順序整理為：

- **01-install.md**  
  從零開始安裝必要工具（VS Code、Git、Claude Code），不假設任何先備知識。

- **02-git-and-github.md**  
  Git 與 GitHub 的新手完整流程：初始化、第一次 commit、SSH 設定、推送到 GitHub。

- **03-project-idea.md**  
  下一個實作專案的構想與規劃，說明如何進入「真正寫程式」的階段，並開始用 Claude Code 建立 React UI。

---

## 適合誰？

- 完全沒有 coding 經驗的新手  
- 對 AI / Claude Code 有興趣，但不知道怎麼開始的人  
- 想把 AI 納入日常開發流程的工程師或產品人  
- 需要做內訓、demo、或 onboarding 教材的團隊

---

## 使用方式建議

1. 建立一個全新的 macOS 使用者帳號（模擬新手）
2. 依序閱讀並實作 `01` → `02` → `03`
3. 照文件實際操作，而不是只閱讀
4. 出現卡關的地方，通常正是教材的重點
